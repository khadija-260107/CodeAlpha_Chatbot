# Basic Chatbot

A rule-based chatbot built with Python as part of the CodeAlpha Python Programming Internship.

## How to Run
python chatbot.py

## Features
- Responds to greetings and general questions
- Tells jokes
- Shows current time and date
- Handles unknown inputs gracefully

## Concepts Used
- Dictionary
- Functions
- Loops
- String handling
